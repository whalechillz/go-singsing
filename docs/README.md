## 📝 문서 작성 규칙

### 1. 날짜 표기
- 모든 날짜는 `YYYY-MM-DD` 형식 사용
- 작업 기록에는 반드시 날짜 포함

### 2. 마크다운 규칙
- 제목은 `#`으로 계층 구분
- 코드는 ` ``` ` 블록 사용
- 리스트는 `-` 또는 `1.` 사용

### 3. 파일명 규칙
- 소문자와 하이픈 사용 (kebab-case)
- 확장자는 `.md` 사용
