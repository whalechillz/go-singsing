# 싱싱골프 투어 관리 시스템

> 60대 여성 시니어를 위한 골프 투어 예약 및 관리 플랫폼

## 🏌️‍♀️ 프로젝트 소개

싱싱골프는 시니어 여성 골퍼들을 위한 맞춤형 골프 투어 서비스입니다.

### 주요 기능
- 📅 투어 일정 관리
- 👥 참가자 관리
- 🏨 숙박 배정
- ⛳ 티타임 관리
- 💰 결제 관리

### 기술 스택
- **Frontend**: Next.js 15, React 19, TypeScript
- **Backend**: Supabase
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## 🚀 시작하기

```bash
# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 배포
./deploy.sh
```
