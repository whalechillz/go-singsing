# 📸 카카오 알림톡 템플릿 이미지 제작 가이드

## 🎨 디자인 규격

### 기본 사양
- **크기:** 800×400px (2:1 비율)
- **형식:** PNG, JPG
- **용량:** 최대 500KB
- **배경색:** #F9F9F9 (밝은 회색)

### 아이콘 설정
- **크기:** 180×180px
- **색상:** #4B5BFF (싱싱골프 브랜드 컬러)
- **위치:** 상단 76px

### 텍스트 설정
- **폰트:** Pretendard Bold 또는 Noto Sans KR Bold
- **크기:** 54pt
- **색상:** #333333
- **최대 글자수:** 9자 (띄어쓰기 포함)

### 드롭 섀도우
- **단순 아이콘** (bed, bus, calendar):
  - Position: X 0, Y 2
  - Blur: 8
  - Spread: 0
  - Color: #000000, 10%
- **복잡 아이콘** (flag, door-open):
  - Position: X 0, Y 4
  - Blur: 4
  - Spread: 0
  - Color: #000000, 25%

## 🖼️ Figma 제작 과정

### 1. 프레임 생성
```
- 크기: 800×400px
- 이름: [템플릿명]_이미지
- 배경색: #F9F9F9
```

### 2. 아이콘 추가 (Iconify 플러그인)
```
1. Iconify 플러그인 실행
2. "Lucide"로 필터링
3. 아이콘 검색 및 선택
4. 색상: #4B5BFF
5. 크기: 180×180px
6. 위치: 중앙, 상단 76px
```

### 3. 텍스트 추가
```
- 텍스트 도구(T) 선택
- 폰트: Pretendard Bold
- 크기: 54pt
- 색상: #333333
- 정렬: 중앙
- 위치: 아이콘 하단 24px
```

### 4. 드롭 섀도우 적용
```
- 아이콘 레이어 선택
- Effects → Drop Shadow 추가
- 설정값 입력 (아이콘 타입별로 다름)
```

### 5. Export
```
- 형식: PNG
- 크기: 1x
- 품질: 100%
- 배경: 포함
```

## 💡 디자인 팁

### DO ✅
- 아이콘 중앙 정렬 유지
- 일관된 스타일 적용
- 충분한 여백 확보
- 명확한 텍스트 가독성
- 브랜드 컬러 준수

### DON'T ❌
- 테두리 추가하지 않기
- 그라데이션 사용 금지
- 복잡한 배경 피하기
- 작은 텍스트 사용 금지
- 여러 색상 혼용 금지

## 🔧 문제 해결

### 아이콘이 흐릿하게 보일 때
- 드롭 섀도우 설정값 확인
- 아이콘 크기가 180×180px인지 확인
- Export 품질 100%로 설정

### 텍스트가 잘 안 보일 때
- 폰트 두께 Bold 확인
- 크기 54pt 확인
- 색상 #333333 확인

### 파일 용량이 클 때
- TinyPNG 등 압축 도구 사용
- 불필요한 레이어 제거
- Export 설정 재확인

## 🚀 빠른 작업 팁

### Component 활용
1. 첫 번째 템플릿을 Component로 만들기
2. 다른 템플릿은 Instance로 복사
3. 아이콘과 텍스트만 변경

### 일괄 Export
1. 모든 프레임 선택
2. Export 설정 동일하게 적용
3. 한 번에 모두 Export
